# Convite digital do Chá Revelação

Arquivos prontos para publicação no GitHub Pages.

## Conteúdo
- `index.html`: site do convite
- `convite.png`: imagem da capa
- `.nojekyll`: impede processamento desnecessário pelo Jekyll

## Publicação
Envie os três arquivos para a raiz de um repositório público e ative:
Settings > Pages > Deploy from a branch > main > /(root)
